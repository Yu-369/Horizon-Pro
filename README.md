# YouTube Pro

Experience YouTube with a stunning cinematic redesign. YouTube Pro transforms the standard interface into a premium, focused, and ultra-modern experience.

![Hero Banner](https://github.com/user-attachments/assets/659217be-ecfd-4f99-ae1b-063d15336f1e)

## Key Highlights

- **Cinematic Carousel**: A dynamic, Google Sans-powered hero section that showcases top content with smooth transitions.
- **Minimalist Aesthetic**: Clean, distraction-free layout removing "div soup" and clutter like the frosted glass bars and chip wrappers.
- **Smart Focus**: Automatically hides irrelevant search shelves, sponsored content, and algorithmic noise to keep you in the flow.
- **Customizable Interface**: Toggle everything from the sidebar density to specific UI elements like Shorts, Comments, or the Notification Bell.

## Installation (Developer Mode)

Since the extension is currently in early access, you can install it using **Developer Mode**:

1. **Download**: Grab the latest release ZIP [here](https://github.com/user-attachments/files/27355650/YouTube.Pro.zip).
2. **Extract**: Unzip the folder to a permanent location on your drive.
3. **Chrome Extensions**: Open Chrome and navigate to `chrome://extensions/`.
4. **Developer Mode**: Toggle the **Developer mode** switch in the top right corner.
5. **Load Unpacked**: Click the **"Load unpacked"** button and select the extracted folder.
6. **Refresh**: Visit [YouTube](https://youtube.com) and enjoy the new experience.

> [!NOTE]
> You may need to refresh any open YouTube tabs after the initial installation.

## Design Philosophy

YouTube Pro is built on the principle of **Intentional Viewing**. We believe the interface should get out of the way of the content. By utilizing modern CSS techniques and subtle JS-driven enhancements, we've created a skin that feels native yet significantly more premium than the standard UI.
